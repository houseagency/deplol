Deplol
======

To publish to Docker registry
-----------------------------

Run:

```
./deploy.sh publish
```

Required environment variables:

* `AWS_ACCESS_KEY_ID`
* `AWS_SECRET_ACCESS_KEY`
* `DOCKER_REGISTRY_HOST`
  (example: `979723745180.dkr.ecr.eu-west-1.amazonaws.com`)
* `DOCKER_REGISTRY_NAME`
  (example: `myserver`)
